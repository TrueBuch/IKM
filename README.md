Учебное web-приложение для управления грузоперевозками, разработанное на ASP.NET Core MVC 
с использованием PostgreSQL и Entity Framework Core.

Приложение позволяет просматривать, добавлять, редактировать и удалять данные о водителях,
грузовиках, грузах, маршрутах и рейсах, а также автоматически отслеживать их статусы.

### Используемые технологии
- ASP.NET Core MVC
- Entity Framework Core
- PostgreSQL
